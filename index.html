Como funciona una red LAN

Funcina interconetando dispositvos a traves de cables Ethernet como el WI-FI o ambo, tambien utilizando switches, routers para dirigiflos de una manera eficiente

Componentes de la red LAN

Sus componenetes son como la tarjeta red, concentradores(hubs), puntos de acceso WI-FI asi como software y sistemas operativos, switches y routers.
